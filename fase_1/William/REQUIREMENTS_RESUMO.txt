PRODUCAO (Servidor, Deploy, sem desenvolvimento):
  pip install -r requirements-prod.txt
  Tamanho: ~500 MB
  Tempo: 3-5 minutos
  Inclui: pandas, numpy, scikit-learn, torch, mlflow, fastapi

DESENVOLVIMENTO (Notebook, Coding, Experiments):
  pip install -r requirements-dev.txt
  Tamanho: ~2 GB
  Tempo: 10-15 minutos
  Inclui: TUDO (dev + prod + ML tools)

USANDO PYPROJECT.TOML (Modern Python):
  pip install .                    (producao)
  pip install -e ".[dev,ml]"       (desenvolvimento)

USANDO MAKE:
  make install-prod                (producao)
  make install-full                (dev completo)
  make help                        (ver todos)

                       COMO INSTALAR
================================================================================

LINUX / MAC:

# 1. Criar ambiente
python -m venv .venv

# 2. Ativar
source .venv/bin/activate

# 3. Instalar
pip install -r requirements-dev.txt

# 4. Verificar
python 02_train_baselines.py

---

WINDOWS (PowerShell/CMD):

# 1. Criar ambiente
python -m venv .venv

# 2. Ativar
.\.venv\Scripts\activate

# 3. Instalar
pip install -r requirements-dev.txt

# 4. Verificar
python 02_train_baselines.py
