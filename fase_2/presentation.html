NCF — Neural CF
User Emb(64d) ─┐ Item Emb(64d) ─┤ Concat └→ Linear(128) → ReLU+Dropout → Linear(64) → Linear(1)→ σ
He et al. (2017) · MLP sobre embeddings
GMF — Gen. MF
User Emb(64d) ─┐ Item Emb(64d) ─┤ Hadamard └→ Product → Dropout → Linear(1) → Sigmoid
Branch element-wise do paper NCF
MF — Matrix Fact.
User Emb + Bias ─┐ Item Emb + Bias ─┤ Dot └→ Product + Biases + Global Bias → Sigmoid
Funk-SVD: μ + b_u + b_i + p·q

Conclusões & Insights

O que aprendemos com 11 modelos e 7 métricas

🏆
Melhor modelo para recomendação
ncf_binary
NDCG@10: 0.3151 HR@10: 0.3650 MRR@10: 0.2962 Rec@10: 0.4492
Métricas de ranking são mais relevantes para recomendação do que AUC-ROC
Binary supera weighted/implicit em ranking
Tratar toda interação como sinal binário de preferência melhora HR@10, NDCG@10 e MRR em modelos NCF — provavelmente por não superestimar a força das views.
MF domina AUC-ROC, NCF domina ranking
Matrix Factorization (implicit) obteve AUC-ROC 0.9299, mas perde para ncf_binary em todas métricas de ranking. Para produção, NDCG e HR são mais relevantes.
GMF binary convergiu mal (Loss: 0.478)
Loss alto sugere que o produto de Hadamard não captura bem interações binárias — a MLP do NCF fornece a não-linearidade necessária para esse tipo de sinal.
